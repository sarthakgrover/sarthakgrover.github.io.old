Precise Indoor Localization using FM

Using USRP2 and BasicTX and BasicRX daughter boards as an RF front-end, we aimed to utilize the RDS frame in commercial FM Radio broadcast for precise indoor localization. RDS band carries precise timing information, and by synchronizing a receiver with multiple RDS transmitters we could calculate its exact location through triangulation. More precision is possible if we also set up a local FM transmitter for accurate timing. Another method we considered was harmonic-based localization where we track the phase of stereo component in an FM broadcast to track a mobile device through its phase difference. This work is currently being adapted to Samsung Handsets for precise indoor localization at NCSU.

Design of Hybrid Wireless Networks for In-Hospital Mobile Sensing Applications

The aim of the project was to design a mesh network using micaZ motes, which could be used in large areas to easily localize and track mobile users and equipments. This work was divided into three phases:
  • Optimal Placement for adequate coverage and connectivity
  • Design of Localization Algorithm using SVM for RSSI Mapping
  • Communication protocol for other tracking and other uses
My research focused on the first part, to design a hybrid multi-hop network consisting of relays and Base Stations (BS). There were two possible placement strategies - placement of nodes in a lattice and random placement. I worked on theoretically modeling the network reliability and delay for these two strategies.
  1. Deterministic placement: Computed closed form expressions for network reliability in hexagonal (lattice) hybrid wireless networks with link failures and fading; proposed tier-rank based routing protocol.
  2. Random placement: Evaluated delivery probability, delay bounds and optimum base station (BS) density in random hybrid networks (Poisson RGG). Modeled network as queue, radius of connectivity as function of BS and relay density, and verified theoretical analysis via simulations on MATLAB.
Applications include mobile patients and equipment tracking and sensing, home health care, baby monitoring, electronic boarding passes, and lost child tracking.

Future work on this project includes information-theoretic modeling of the hybrid network and exploring the relationship between base station density and network reliability in random networks.

Wireless Body Area Networks for Athlete Monitoring

The aim of this project was to study the statistical properties of the social network (connectivity pattern) between players during a soccer game. The scenario had 11 players in a team with motes attached on their arms, and base stations placed around the field. Due to the size and power of motes, multi-hop routing was needed to relay messages from players in the center of the field to the sideline where base stations were placed.

 
 


We monitored inter-connectivity between players using motes at first, and wireless bio-medical sensors (by Toumaz Technology) later. My goal was to analyze the time-series experimental data and developed encounter based model to capture player mobility. By finding out the exact statistical properties of player-encounters, we generated synthetic network topologies. We found that certain links are highly correlated (inter-encounter times and duration) whereas others are anti-correlated.

The main motivation for this work was to design and simulate a real-time routing scheme for dynamic multi-hop networks observed during field games. Using correlation information between user links, an efficient routing mechanism could be developed to relay a message from any player to the base station more efficiently than arbitrary flooding. We simulated one such protocol using MATLAB.

Applications of such correlation-based mobility models include providing referee-assist and enhanced television broadcast services during games. This work was published at IEEE WiMob 2010, Niagara Falls, Canada and was awarded the best student paper (download here).

Real Time Human Face Tracking System

NDTV GoodTimes - Face Tracking System
This work required us to develop a robust real-time face tracking algorithm along with the hardware components which could utilize this. Using rank deficient face detection (for its faster speed of execution) we developed a scheme to iteratively search and track the human face in each video frame. Furthermore, this system was demonstrated mounting a web camera on robotic motorized arm with 3 degrees of freedom. The arm was controlled via the algorithm implementation on MATLAB on a desktop using RS-232 cable. The motivation for this work was to focus on speaker during video lecture recordings at IIT, conferences etc.

A demonstration of our work was broadcast on television as 'The Next Big Thing' and results were published in proceedings of IEEE CIS 2008 (download here).

Performance Evaluation of a Wireless Body Area Networks

This work dealt with the study and evaluation via simulation of various protocols on intra-BAN and extra-BAN. Intra-BAN refers to the network formed between nodes placed in or on a human body using the human-body itself as a channel for communication. In contrast, the extra-BAN refers to the body area network between two nodes communicating through air via the wireless channel.
  • Considering intra-BAN, we modeled the human body channel and tested various available protocols (802.11, TDMA, SMAC etc) for energy and throughput efficiency.
  • To consider extra-BAN channel, we assumed mobile users and a Raleigh Fading environment. Then, we modeled the received signal in such a scenario as a Markov chain and autoregressively predicted the state of the channel (in fading or not)
  • Utilizing predicted information at PHY layer we designed and simulated a cross layer protocol based on IEEE 802.11 RTS/CTS to control packet queue at MAC level. Performed comprehensive simulations on NS2 and showed improvements in network throughput and lifetime for a MANET.
Applications include energy eﬃcient athlete monitoring and remote health care for mobile users.

Implementation of a Soft Decision Decoder using Trellis on FPGA

As part of my Master's project, I implemented a real-time trellis decoder for BCH codes and configured it on FPGA. This work was completed using VHDL on Xilinx ISE software. Dedicated hardware was shown to achieve higher data rate compared to a software implementation.

Alarm Handling for Nokia Epsilon Base Transceiver System

Project included a critical analysis of GSM Networks and Protocols and BTS Controller tasks. We developed an alarm handling functional module for integration with System Operations & Maintenance (SOM) processes of the Nokia Epsilon BTS.

Design of Circular Patch Micro-strip Antenna using PSO

The goal of this work was to utilize Particle Swarm Optimization (PSO) for dimensional analysis of a circular patch antenna. The objective function includes the resonant frequency in the dominant TM11 mode with constraints which include limits on the radius of patch, thickness and relative permittivity of the substrate. We showed that PSO requires less time to achieve the same dimensional accuracy as compared to Genetic Algorithm.
